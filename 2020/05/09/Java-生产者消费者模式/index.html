Java 生产者消费者模式

生产者消费者模式将产生数据的执行流叫做生产者，将处理数据的执行流叫做消费者，为生产者于消费者提供数据缓冲区，生产者将数据写入缓冲区，消费者从缓冲区中获取数据，实现生产者于消费者的同步。

优点

  • 优点

    • 降低耦合度

      由于有缓冲区的概念，生产者与消费者之间没有直接的依赖关系，可以降低系统耦合度

    • 提高效率

      生产者与消费者并发允许，操作数据缓冲区，提高了系统效率

    • 平衡生产者与消费者之间的处理速度差
